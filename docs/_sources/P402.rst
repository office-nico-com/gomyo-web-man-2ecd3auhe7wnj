{CHAPTER_LV2} 入金処理
====================================================================

| 入金日や入金金額を登録します。
| 登録した内容は入金合計として集計され、請求残高や未収金の管理に使用されます。


入金情報の登録
--------------------------------------------------------------------

工事情報詳細画面上部の「入金入力」ボタンをクリックしてください。登録フォームが開きます。
入金日や入金額などを入力し、「保存」ボタンをクリックしてください。


.. image:: _static/402.png
   :width: 700px


登録した内容は、工事情報詳細画面の入金情報エリアに表示されます。これらの情報は入金合計として集計され、請求残高および未収金の管理に使用されます。

.. image:: _static/405.png
   :width: 500px


請求情報の編集
--------------------------------------------------------------------
入金情報の行をクリックすると編集フォームが開きます。内容を修正後、「保存」ボタンをクリックしてください。


請求情報の削除
--------------------------------------------------------------------
入金情報の行の先頭に表示されているゴミ箱アイコンをクリックしてください。確認ダイアログが表示されますので、「削除」をクリックするとデータが削除されます。

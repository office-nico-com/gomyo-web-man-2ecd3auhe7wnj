{CHAPTER_LV2} 請求処理
====================================================================

| 請求日や請求金額などの請求情報を登録します。
| 登録した内容は請求合計として集計され、契約金額との比較や売上管理に使用されます。


請求情報の登録
--------------------------------------------------------------------

工事情報詳細画面上部の「請求入力」ボタンをクリックしてください。登録フォームが開きます。
請求日や請求額などを入力し、「保存」ボタンをクリックしてください。


.. image:: _static/402.png
   :width: 700px


登録した内容は、工事情報詳細画面の請求情報エリアに表示されます。これらの情報は請求合計として集計され、売上管理および未請求残高の把握に使用されます。

.. image:: _static/403.png
   :width: 500px


請求情報の編集
--------------------------------------------------------------------
請求情報の行をクリックすると編集フォームが開きます。内容を修正後、「保存」ボタンをクリックしてください。


請求情報の削除
--------------------------------------------------------------------
請求情報の行の先頭に表示されているゴミ箱アイコンをクリックしてください。確認ダイアログが表示されますので、「削除」をクリックするとデータが削除されます。

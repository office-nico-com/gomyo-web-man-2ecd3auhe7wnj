{CHAPTER_LV2} 社員マスタ
====================================================================

| 社員マスタでは、工事に従事する社員情報を登録します。登録した社員は、労務費入力時に選択して使用します。
| 社員情報をあらかじめ登録しておくことで、入力の効率化と正確な労務費集計が可能になります。

起動
--------------------------------------------------------------------

左メニューから「社員マスタ」のアイコンをクリックしてください。登録済みの社員一覧が表示されます。

.. image:: _static/105.png
   :width: 700px


社員の登録
--------------------------------------------------------------------

社員マスタ画面の右上にある「新規登録」ボタンをクリックしてください。登録フォームが開くので必要事項を入力して「保存」ボタンをクリックしてください。

.. image:: _static/106.png
   :width: 700px


社員の編集
--------------------------------------------------------------------

社員マスタの行をクリックすると編集フォームが開きます。内容を修正後、「保存」ボタンをクリックしてください。


社員の削除
--------------------------------------------------------------------


社員マスタの行の先頭に表示されている「ゴミ箱アイコン」をクリックしてください。確認ダイアログが表示されるので、「削除」をクリックすると、データが削除されます。

.. image:: _static/107.png
   :width: 400px

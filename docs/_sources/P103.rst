{CHAPTER_LV2} メーカー・品名マスタ
====================================================================

| 材料費入力時に選択できるメーカーおよび品目情報を登録します。
| あらかじめ登録しておくことで入力を効率化できますが、材料費入力時に直接入力することも可能です。


起動
--------------------------------------------------------------------

左メニューから「メーカー・品名マスタ」のアイコンをクリックしてください。登録済みのメーカー・品名一覧が表示されます。

.. image:: _static/108.png
   :width: 700px


社員の登録
--------------------------------------------------------------------

メーカー・品名マスタ画面の右上にある「新規登録」ボタンをクリックしてください。登録フォームが開くので必要事項を入力して「保存」ボタンをクリックしてください。

.. image:: _static/109.png
   :width: 700px


社員の編集
--------------------------------------------------------------------

メーカー・品名マスタの行をクリックすると編集フォームが開きます。内容を修正後、「保存」ボタンをクリックしてください。


社員の削除
--------------------------------------------------------------------


メーカー・品名マスタの行の先頭に表示されている「ゴミ箱アイコン」をクリックしてください。確認ダイアログが表示されるので、「削除」をクリックすると、データが削除されます。

.. image:: _static/110.png
   :width: 400px
